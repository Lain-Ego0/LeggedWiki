# 推荐论文（本地 PDF）

该目录收录论文/讲义/笔记等 PDF（优先保存可公开获取版本，例如 arXiv 预印本）。

- 重点论文索引/笔记：见 [reading-list/README.md](<../reading-list/README.md>)
- 如需批量抓取可公开 PDF：更新 `reading-list/papers.json` 后运行 `python3 scripts/fetch_open_access_pdfs.py --download`

## 目录结构
- ETH RSL：`recommended-papers/ETH-RSL/`
- KAIST：`recommended-papers/KAIST/`
- MIT：`recommended-papers/MIT/`
- 中文资料（学位论文/笔记等）：`recommended-papers/chinese/`

- 人形：`recommended-papers/humanoid/`
